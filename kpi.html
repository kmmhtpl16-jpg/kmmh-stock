⚙️ ตั้งค่าโบนัส B / C (เจ้าของกรอก/แก้เองได้)

••••
••••
วิธีจ่าย B/C: เคาะตอนปิดรอบ (B = สิ้นไตรมาส · C = สิ้นปี) = ส่วนสัดส่วน (จ่ายตาม %หมวดที่นับ) + ส่วนปิดครบ (จ่ายเมื่อครบ 100%) · ตั้งสัดส่วน:ปิดครบ = 50:50
ประมาณการตอนนี้: B นับ 0% → โบนัส •••• · C นับ 0% → โบนัส ••••
ทยอยส่งได้ จ่ายตามที่นับได้จริงตอนปิดรอบ · ทำครึ่งทางแล้วทิ้ง = ได้แค่ส่วนสัดส่วน ไม่ได้ส่วนปิดครบ
